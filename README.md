# Orbit: Kinetic Defense – Pro Edition
> **A high-performance, physics-based space defense game for Web & Mobile.**

`Orbit: Kinetic Defense` is an arcade-style physics puzzler built with **HTML5 Canvas** and **JavaScript**. It features a "drag-to-launch" mechanic where players deploy satellites to neutralize incoming threats using orbital trails.

### **🚀 Key Features**
* **Physics-Driven Combat:** Intuitive drag-to-aim launching system.
* **Level Progression:** 20 handcrafted sectors with unique gravity layouts.
* **In-Game Economy:** Full upgrade shop using **Credits (💎)**.
* **Mobile Ready:** Optimized for touch controls and Android deployment.

### **🛠 Technical Architecture**
* **Engine:** Vanilla JavaScript / HTML5 Canvas.
* **Persistent Progress:** Automated save states using `localStorage`.
